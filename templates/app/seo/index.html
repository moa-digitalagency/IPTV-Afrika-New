À Propos de SEO

Les meta tags aident les moteurs de recherche et réseaux sociaux à comprendre vos pages. Chaque page peut avoir des meta titles, descriptions, et tags OpenGraph uniques.

Meta Title

Titre affiché dans les SERP (max 60 chars)

Meta Description

Description affichée sous le titre (max 160 chars)

Open Graph

Contrôle l'apparence lors de partages réseaux

Robots

Directives pour crawlers (index, follow, noindex)
